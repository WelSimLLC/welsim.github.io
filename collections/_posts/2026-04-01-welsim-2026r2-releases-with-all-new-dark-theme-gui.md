WELSIM, a general-purpose engineering simulation and analysis software, has released its latest 2026R2 version (internal version 3.3). Compared with the previous release, version 2026R2 delivers a major upgrade to its underlying framework, while introducing a new dark theme, making the graphical user interface more user‑friendly.
<p align="center">
  <img src="\assets\blog\20260401\welsim_release_2026r2.png" alt="welsim_release_2026r2" />
</p>

## All new dark theme
The newly added dark theme follows the trend of modern desktop applications and is gentler on users' eyes. Users can set the appearance theme in Preferences, with three options available: Classic, Dark, and Light.
<p align="center">
  <img src="\assets\blog\20260401\welsim_thermal_ss_dark.png" alt="welsim_thermal_ss_dark" />
</p>


The Classic theme is similar to the Light theme, whereas the Dark theme features a dark background with light‑colored text and icons in the foreground. Beginning with this version, the Dark theme is set as the default style for WELSIM.

## Comprehensive Framework Upgrade
The new version has upgraded its key foundational frameworks, including updating the GUI framework Qt from 5.15 to 6.10, and the 3D rendering engine VTK from 8.2 to 9.2.
<p align="center">
  <img src="\assets\blog\20260401\welsim_about.png" alt="welsim_about" />
</p>

These cross‑major‑version upgrades of the two core frameworks have improved the software's stability and performance.


## Other Improvements and Upgrades
In addition, the new release optimizes and enhances existing features and adds more automated test cases, making WELSIM more stable and easier to use.
