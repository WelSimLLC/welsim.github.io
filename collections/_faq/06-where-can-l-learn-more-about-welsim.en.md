title: "Where can I learn more about WELSIM?"
---

The ease-to-use GUI makes WELSIM nearly no learning curve. You can start with our documentation at [docs.welsim.com](https://docs.welsim.com). In addition, the automatic regression tests are available at [https://github.com/WelSimLLC/WelSimAutoTests](https://github.com/WelSimLLC/WelSimAutoTests), You can save the test cases that you have run to obtain sample projects.
